# git 使用

## 常用命令

* git init 初始化 git 目录
* git status 查看目录当前 git 管理状态
* git add 将一个文件添加给 git 管理
* git diff 对比修改内容
* git commit 提交修改
* git remote 管理远程仓库
* git clone 将远程仓库克隆到新目录中
* git pull 拉取远程仓库
* git push 推送本地版本到远程仓库

.gitignore 忽略管理列表

## 问题

在 vscode 中使用 git 链接 github ，提示没有权限

> 暂时的解决方案，把 git 链接改成 https。
>
> 补充: 或者在生成秘钥的时候不去设置秘钥密码。
